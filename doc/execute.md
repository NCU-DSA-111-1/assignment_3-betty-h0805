# Execute Guide
## Huffman
進入 `bin` 資料夾
```
cd bin
./huffman encode(or decode) <input filename> <output filename>
```

![](https://i.imgur.com/rn1UF8B.png)

## Arithmetic
進入 `arcd` 資料夾
-c 為 encode
-d 為 decode
```
cd arcd
./arcd (-c or -d)-i <input filename> -o <input filename>
```
